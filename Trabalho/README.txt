Para realizar o uso do programa é necessario um compilador em C,
o que foi usado para a implementação e teste foi o GCC

O formato do arquivo deve ser respeitado desta seguinte forma

pode_ir(S1, S3, DISTANCIA)
pode_ir(S2, S3, DISTANCIA)

Respeitando o S acompanhado do numero da sala, por isso S1 (sala 1)
assim como os espacos depois da virgula e a distancia nao pode ser negativa

O arquivo que foi usado como teste está na pasta também com o nome de sala.txt

Não é possivel realizar alguma operação antes de carregar o arquivo, para carregar o arquivo
digite 1 para digitar o nome do arquivo levando em conta o .txt, então para entrar com o arquivo
terá que ser digitado da seguinte forma "sala.txt"
